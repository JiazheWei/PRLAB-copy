# Research Proposal: Dynamic Memory for Physically-Consistent World Models in Open-Ended Environments

**Dynamic Memory for Physically-Consistent World Models in Open-Ended Environments**

现有的视频生成世界模型（world model）虽然已展现出一定的场景记忆能力，但这种记忆本质上是**静态的（static memory）**——即模型能够在视角切换后保持场景的几何与外观一致性（例如，摄像机移开再移回时，桌子的形状和位置不变）。然而，真实世界中的记忆远不止于此：当智能体暂时未观测到某一区域时，该区域内的物理过程仍在持续演化。例如，一个人转头后，身后另一人正在向杯中倒水；当该人再次回头时，杯中水位必须相应升高。我们将这种对**不可见区域中物理状态持续演变的建模与追踪**能力定义为**动态记忆（dynamic memory）**。

当前的世界模型普遍缺乏这一能力，导致其在生成长时程、多视角交互内容时出现物理不一致的问题。为解决这一挑战，本研究提出在 **Minecraft（MC）** 这一结构化、可控且物理规则明确的开放世界环境中，系统性地研究世界模型的动态记忆机制。Minecraft 提供了理想的实验平台：其离散化的方块世界便于精确定义和量化物理状态变化，同时场景的开放性和交互复杂度足以体现动态记忆的核心难点。

在数据层面，我们计划基于谢赛宁团队提出的 **Solaris** 框架构建训练与评测数据集。Solaris 提供了大规模、高质量的 Minecraft 环境交互数据，我们将在此基础上设计包含**离屏物理事件（off-screen physical events）**的数据采集协议，系统性地构造涵盖物质流动、物体位移、状态变迁等多类动态变化的场景对，从而为动态记忆能力的学习与评估提供坚实的数据基础。

本研究旨在推动世界模型从"静态快照式记忆"迈向"动态物理一致性记忆"，为构建真正理解并预测世界运行规律的通用世界模型奠定基础。
